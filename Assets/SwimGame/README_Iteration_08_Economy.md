# README — Iteration 08 — Economy: Coins, Rewards & Shop

## 1. Что было добавлено
- **Монеты.** CurrencyManager (PlayerPrefs "coins") с событием изменения баланса. Счётчик с иконкой-монетой в top bar игровой сцены и на экране LEVELS
- **Награда за победу.** Улучшил звёзды на уровне — 10 + 5 за каждую звезду (3 звезды = 25 монет). Перепрошёл без улучшения — 5 монет. Награда видна в WinPopup ("+25")
- **Анимация монет.** После звёзд из попапа вылетает стайка монет (разлёт OutQuad, слёт InQuad со стаггером), каждая прилетевшая монета — звук Coin и punch иконки, счётчик докручивается через DOVirtual.Int. Если закрыть попап раньше — награда зачисляется мгновенно (без потерь, есть страховка в OnDestroy)
- **Магазин.** Попап Shop: Flippers 30, Ball 40, Shield 50 монет. Покупка = +1 предмет навсегда. Кнопки тускнеют, если не хватает монет (тап — отказ + тряска баланса). Открывается кнопкой "$" в ItemBar или тапом по предмету с x0
- **Персистентный инвентарь.** ItemInventory (PlayerPrefs "item_flippers"/"item_ball"/"item_shield"). Дебаг-выдача x3 осталась галкой Debug Grant Items на ItemManager — теперь ВЫКЛЮЧЕНА по умолчанию (сессионная, не трогает сохранения)

## 3. Editor скрипты — какие меню запустить и в каком порядке
1. `SwimGame -> Update Game Scene (Iteration 8)` — монета+счётчик в top bar, кнопка "$" в ItemBar (предметные кнопки сдвигаются влево), попап магазина, "+N" в WinPopup, все ссылки
2. `SwimGame -> Update Level Select Scene (Iteration 8)` — счётчик монет на экране LEVELS

## 4. Как тестировать
1. Пройти уровень впервые на 3 звезды — в попапе "+25", монеты разлетаются и слетаются к счётчику, счётчик докручивается
2. Нажать NEXT во время полёта монет — баланс всё равно корректный (зачисление мгновенное при закрытии)
3. Перепройти уровень с теми же звёздами — "+5"; улучшить с 1 до 3 звёзд — "+25"
4. Открыть магазин кнопкой "$": купить Flippers за 30 — баланс уменьшился, в ItemBar x1; не хватает монет — кнопка тусклая, тап даёт отказ и тряску баланса
5. Перезапустить игру — монеты и предметы на месте
6. Тап по предмету с x0 — открывается магазин
7. Включить Debug Grant Items на ItemManager — на старте уровня по 3 предмета (сессионно, PlayerPrefs не трогаются)

## 5. Ожидаемый результат
Полная петля экономики: проходишь уровни -> копишь монеты -> покупаешь предметы -> проходишь сложные уровни на 3 звезды. Каждое начисление и покупка озвучены и анимированы.

## 6. Известные ограничения текущей итерации
- Иконки монеты и предметов — кружки/буквы-плейсхолдеры
- Магазин доступен только из игровой сцены (с экрана LEVELS — в следующих итерациях при необходимости)
- Баланс наград/цен первичный, крутится константами в LevelLoader и ShopPopup
