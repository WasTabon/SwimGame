# README — Iteration 09 — Settings, Pause & Tutorial

## 1. Что было добавлено
- **Попап настроек.** Три переключателя: SOUND / MUSIC / VIBRATION — ON (оранжевый) / OFF (тусклый) с punch-анимацией и мгновенным применением (SoundManager/HapticManager, сохраняется в PlayerPrefs). Открывается из главного меню кнопкой SETTINGS и из паузы в игре
- **Пауза.** Кнопка "II" в top bar (Moves сдвинут левее). PausePopup: RESUME, RESTART, SETTINGS (открывается поверх паузы), MENU (в LevelSelect). Время не трогаем — игра пошаговая, бекдроп блокирует игровой инпут
- **Туториал на уровнях 1-3.** Без текста: над клеткой следующего хода пульсирует оранжевый указатель (решения из солвера итерации 7). Показывается только если на уровне ещё 0 звёзд. Сделал ход по подсказке — указатель переходит дальше; свернул с маршрута — исчезает до рестарта (свобода не ограничивается). На победе/поражении скрывается

## 4. Как тестировать
1. Главное меню -> SETTINGS: выключить SOUND — тапы замолкают сразу; выключить MUSIC — музыка стихает; настройки переживают перезапуск
2. В игре нажать "II": пауза, тапы по полю не проходят (бекдроп); RESUME возвращает, RESTART перестраивает уровень, SETTINGS открывает настройки ПОВЕРХ паузы (закрытие возвращает в паузу), MENU уводит в LEVELS
3. Сбросить PlayerPrefs и зайти в уровень 1: пульсирующий указатель ведёт по маршруту D D D D R R R; следовать ему до конца — 3 звезды
4. В уровне 2 нарочно сходить мимо подсказки — указатель исчезает, играешь свободно; RESTART возвращает подсказки
5. Пройти уровень 1, перезайти — указателя больше нет (звёзды > 0)
6. Туториал не мешает: WAIT, предметы, рестарты работают как раньше

## 5. Ожидаемый результат
Полный продуктовый каркас: настройки доступны из меню и из паузы, новичка ведут первые три уровня, всё анимировано и озвучено.

## 6. Известные ограничения текущей итерации
- Указатель туториала не подсказывает WAIT и предметы (решения уровней 1-3 их не требуют)
- Иконка паузы — текст "II" (заменится ассетом)
