# README — Iteration 06 — Items: Flippers, Ball, Shield

## 1. Что было добавлено
Система предметов с инвентарём в HUD (3 квадратные кнопки F/B/S со счётчиками над WAIT/RESTART):
- **Flippers (F, бирюзовый)** — тап по кнопке включает режим прицеливания: подсвечиваются клетки на расстоянии 2 по прямой (обе клетки пути должны быть проходимы). Тап по цели — рывок на 2 клетки (последовательно, с проверками на каждой: NPC на промежуточной клетке = смерть без щита, выход на любой клетке = победа). Рывок = 1 ход, расходует предмет. Повторный тап по кнопке — отмена без траты
- **Ball (B, оранжевый)** — режим прицеливания: подсвечиваются проходимые клетки в радиусе 4. Тап — мяч летит по дуге, бросок = 1 ход. Следующие 3 World Phase ВСЕ NPC в радиусе 3 от мяча игнорируют своё поведение и делают 1 шаг к мячу (включая Delayed и Fast). Потом мяч тонет. Приманенный NPC всё ещё убивает, если его шаг к мячу попадает на тебя
- **Shield (S, голубой)** — мгновенное использование = 1 ход. Щит (пульсирующая аура) действует на этот ход + следующий: все столкновения исключены — NPC и течения не могут затолкнуть никого в клетку друг друга, а клетки с NPC становятся недоступны для твоего хода/рывка (никаких самоубийств под щитом)

Предметы пока выдаются в дебаг-режиме: по 3 каждого на старте уровня (галка Debug Grant Items на ItemManager). Невалидный тап в режиме прицеливания — звук отказа и выход из режима.

## 4. Как тестировать
1. Любой тестовый уровень: на старте у каждого предмета x3
2. **Flippers:** тап F — бирюзовые маркеры на дистанции 2; рывок через коридор; попробовать рывок через клетку с NPC (смерть на промежуточной); тап F повторно — отмена без траты
3. **Ball:** тап B — оранжевые маркеры радиуса 4; бросить рядом с NPC — он 3 хода идёт к мячу (стрелка поворачивается к мячу), потом мяч тонет с всплеском и NPC возвращается к своему поведению
4. **Shield:** активировать рядом с NPC и пойти прямо на его маршрут — NPC упирается и не входит в твою клетку; заметить, что клетки С NPC перестают приниматься (звук отказа); через ход после активации аура исчезает
5. Потратить все 3 предмета — кнопка тускнеет, тап даёт отказ
6. RESTART — предметы снова x3, мяч/щит/прицеливание сброшены

## 5. Ожидаемый результат
Кнопка активного режима прицеливания увеличена (scale 1.12). Маркеры прицеливания пульсируют. Мяч летит по дуге с всплеском при приводнении и покачивается на воде. Щит — полупрозрачный голубой круг, пульсирующий вокруг игрока. Каждое использование предмета тратит ровно 1 ход и запускает полный World Phase.

## 6. Известные ограничения текущей итерации
- Предметы не сохраняются между уровнями — дебаг-выдача по 3 (магазин и персистентность в итерации 8)
- Притяжение мяча упрощает всех NPC до 1 шага/ход (намеренно, для просчитываемости)
- Иконки предметов — буквы-плейсхолдеры F/B/S (заменятся ассетами)
